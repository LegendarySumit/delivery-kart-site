## ğŸš€ Getting Started

### Prerequisites

No dependencies required! This is a pure frontend application.

### Installation

1. Clone the repository
```bash
   git clone https://github.com/legendarysumit/delivery-kart-site.git
```

2. Navigate to the project directory
```bash
   cd delivery-kart-site
```

3. Open `index.html` in your browser
```bash
   # On macOS
   open index.html
   
   # On Windows
   start index.html
   
   # On Linux
   xdg-open index.html
```

Alternatively, use a local development server:
```bash
# Using Python 3
python -m http.server 8000

# Using Node.js (http-server)
npx http-server
```
